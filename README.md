# Бот для онлайн‑школы: расписание и баллы

Интерактивный Telegram‑бот для онлайн‑школы с функцией выдачи расписания уроков и системой начисления баллов за активность.

## Особенности

* Интерактивное меню с кнопками для удобного взаимодействия.
* Расписание уроков на неделю с наглядным форматированием.
* Система баллов: 100 баллов за каждый запрос расписания.
* Уровни пользователей: Новичок → Профи → Эксперт.
* Дружелюбный интерфейс с мотивирующими сообщениями.
* Простота настройки и расширения функционала.

## Технологии

* Язык программирования: Python 3.x
* Библиотека для Telegram API: pyTelegramBotAPI (`telebot`)
* Хранение данных: словарь в памяти (для продакшена рекомендуется БД)
